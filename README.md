📄 Project Description This Digital Literacy Project focuses on understanding and applying essential digital skills required in today’s world. The project includes creating an infographic to spread awareness, building a professional digital portfolio using platforms like GitHub, LinkedIn, and Kaggle, and exploring coding and collaboration tools such as HackerRank and Google Forms. It also involves learning professional communication by writing formal emails and understanding proper digital etiquette. Additionally, the project covers cybercrime awareness, specifically phishing, and provides safety measures to protect against online threats. Overall, this project helped in improving digital knowledge, communication skills, and awareness about safe and responsible use of technology.
